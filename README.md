# Event Management - Event Management System with category filters, venue â†’ dates mapping, booking, payment simulation, and admin panel.

Features:
    User registration and login
    Admin and regular user roles
    Browse events by category
    View venue-specific available dates for events
    Book events and simulate payment
    Admin dashboard to:
    Approve or reject bookings
    Add, edit, delete events
    View all users and activity logs
    Generate PDF receipts after payment
    Stats page for total users, revenue, recent activity
    Password change functionality
    Fully responsive front-end (Flask + Jinja2)

## Quick start
1. python -m venv venv
2. venv\Scripts\activate
3. pip install -r requirements.txt
4. python app.py
5. Open http://127.0.0.1:5000

Default admin: admin@events.local / admin123
